# Text Animation: Cascade Up

Fast, energetic word-by-word reveal with upward motion.

## Parameters

```yaml
name: cascade-up
type: text
stagger_delay: 0.06
opacity_duration: 1.2
movement_duration: 1.4
initial_y: 60
final_y: 0
ease: [0.34, 1.56, 0.64, 1]
```

## Description

Each word animates independently with a slight delay:
- Words start 60px below center, fade in while accelerating upward
- Stagger delay of 0.06s creates a cascading wave effect
- Custom easing with slight bounce at the end
- Fast and dynamic, perfect for impactful statements

## Visual Effect

Words flow in rapidly, one after another, with smooth upward motion and a subtle bounce.

## Best For

- Short, punchy statements
- Key messages
- Title slides
- High-energy moments
